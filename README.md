# N-Body Problem Simulator
N体問題を解くためのPythonプログラムを作る。

## N体問題とは
相互作用する複数の物体の挙動は、解析的に求められないということが分かっている。
そこで、数値計算によって求める。

## 進捗
一応計算の実装は完了。
